Проект создан в образовательных целях для изучения основ машинного обучения и оптимизации.
Я сделал два файла по градиентному спуску для линейной регрессии

Этот проект демонстрирует реализацию алгоритма градиентного спуска для обучения линейной регрессии с нуля на Python.

## Описание

Проект содержит два примера применения градиентного спуска:
1. **Предсказание цены яблок** - на основе веса (кг) предсказывается цена
2. **Предсказание времени доставки** - на основе расстояния (км) предсказывается время доставки в минутах

## Техническая реализация

- Модель: Линейная регрессия `y = w * x + b`
- Алгоритм обучения: Градиентный спуск (Batch Gradient Descent)
- Функция потерь: Среднеквадратичная ошибка (MSE) и Абсолютная ошибка (MAE)
- Оптимизация: Минимизация MSE и MAE через вычисление градиентов и оптимизации по формуле w = w - скорость обучения * градиент

## Вывод программы
На каждой итерации выводится:

\n-Номер итерации
\n-Текущие значения w и b
\n-Предсказания модели
\n-Ошибки предсказаний
\n-Значение функции потерь (MSE/MAE)
\n-Обновленные параметры после шага градиентного спуска
